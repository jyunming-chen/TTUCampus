# TTU Campus

[![build](https://img.shields.io/travis/jyunming-chen/TTUCampus/master.svg)](https://travis-ci.org/jyunming-chen/TTUCampus/builds)

## Usage

```sh
npm install
npm run build
npm run start
```

## Development

- node >= 6

```sh
# lint
npm run lint

# build
npm run build:dev

# start
npm run start:dev

# test
npm run test
```
